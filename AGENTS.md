# Ubermench — AI Agent Context

This project uses a universal context system in `.sanctuary/`.
Start by reading `.sanctuary/ONBOARDING.md`.
Then read `.sanctuary/BITACORA.md` (last 3-5 entries) — the cross-tool session log. It tells you what changed since you last looked. Log your own session there if it changes something significant.

## Tool-Specific Files

- **Claude Code:** `CLAUDE.md`
- **Cursor:** `.cursor/rules/`
- **Windsurf:** `.windsurfrules`
- **GitHub Copilot:** `.github/copilot-instructions.md`

If your tool is not listed, read `.sanctuary/ONBOARDING.md` directly.
